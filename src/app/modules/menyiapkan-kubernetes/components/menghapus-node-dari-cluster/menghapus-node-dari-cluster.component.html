5. Menghapus Node dari Cluster

Berikut adalah tahap untuk menghapus node dari cluster. Pertama, node yang ingin dihapus harus di-drain terlebih dahulu. Drain adalah proses menghentikan semua pods yang berjalan di node tersebut. Jalankan perintah berikut

xxxxxxxxxx
kubectl drain <node_name> --ignore-daemonsets --delete-local-data

Setelah selesai, hapus node dengan cara

xxxxxxxxxx
kubectl delete node <node_name>

Node telah terhapus dari cluster tetapi kubeadm masih aktif di node tersebut. Buka terminal node tersebut dan jalankan perintah berikut

xxxxxxxxxx
kubeadm reset

Kemudian, untuk membersihkan sisa-sisa berkas dan konfigurasi yang masih tersimpan di node tersebut, lakukan perintah berikut

rm -rf /var/lib/cni;
rm -rf /var/lib/kubelet/*;
rm -rf /run/flannel;
rm -rf /etc/cni;
ifconfig cni0 down & ip link delete cni0;
ifconfig flannel.1 down & ip link delete flannel.1;
brctl delbr cni0;
brctl delbr flannel.1;
