1. Node

Node
adalah suatu objek dari Kubernetes yang merepresentasikan satu mesin dari cluster tempat menjalankan containers.

Terdapat dua jenis dari node:

  1. master node, yang mengatur alur perintah dari pengguna menuju cluster;
  2. worker node, tempat menjalankan containers dari perintah pengguna;

Setiap node memiliki nama berdasarkan hostname yang diberikan saat membuat cluster. Node juga dapat diberi label yang berguna untuk menandai node mana tempat container ingin dijalankan.

Untuk melihat daftar node, jalankan perintah

xxxxxxxxxx
kubectl get nodes

Keluarannya kira-kira akan seperti ini

xxxxxxxxxx
NAME         STATUS   ROLES    AGE   VERSION
master-240   Ready    master   28m   v1.19.2
master-241   Ready    master   22m   v1.19.2
worker-55    Ready    <none>   20m   v1.19.2
worker-56    Ready    <none>   20m   v1.19.2

Untuk memberi label pada node, jalankan perintah

xxxxxxxxxx
kubectl label nodes <node_name> <label_key>=<label_value>

Misalnya, pengguna ingin memberi label pada node dengan nama worker-nginx untuk menandai node tersebut hanya untuk menjalankan aplikasi nginx. Maka, perintahnya adalah

xxxxxxxxxx
kubectl label nodes worker-nginx app=nginx

Setiap node bisa memiliki banyak label, pengguna dapat menambah label sesuka hati

Untuk menghapus label, pengguna dapat menjalankan perintah

kubectl label nodes <node_name> <label_key>-
