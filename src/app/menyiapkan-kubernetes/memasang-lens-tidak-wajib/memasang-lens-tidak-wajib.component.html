6. Memasang Lens (tidak wajib)

Aplikasi Lens adalah sebuah aplikasi yang memudahkan pengguna untuk mengakses cluster dan mengirim perintah ke dalamnya tanpa harus repot-repot mengetiknya ke dalam terminal.

Saat memulai aplikasi Lens, pengguna akan diminta untuk memasukkan berkas config Kubernetes untuk memilih context yang ingin digunakan (atau memilih semua context yang nanti bisa dengan mudah berganti dengan sekali klik).

Berikut adalah tampilan awal dari aplikasi Lens

Secara default, Lens akan mencari berkas config Kubernetes di dalam $HOME/.kube/config. Jika ingin menggunakan berkas config lain, silakan pilih ikon browse untuk memilih berkas lain. Setelah itu, pilih context yang akan digunakan, dan tekan tombol Add cluster(s).

Akan muncul tampilan seperti berikut

Terlihat bahwa di sebelah kiri atas terdapat ikon yang merupakan context yang telah dipilih. Dengan menekan ikon tersebut, pengguna dapat langsung menggunakan context tersebut. Berikut adalah tampilan awal saat context dibuka

Jika tampilanmu tidak seperti pada gambar, hal itu karena belum terpasangnya kube-metrics dan prometheus dalam cluster. Pilih menu File pada menu bar dan pilih Cluster Settings sehingga muncul tampilan seperti berikut
